eric6.Plugins.CheckerPlugins.SyntaxChecker

Package containing the Syntax Checker plugin.

Modules

SyntaxCheck Module implementing the syntax check for Python 3.
SyntaxCheckService Module implementing an interface to add different languages to do a syntax check.
SyntaxCheckerDialog Module implementing a simple Python syntax checker.
jsCheckSyntax Module implementing the syntax check for JavaScript.
jsonCheckSyntax Module implementing the syntax check for JSON.
tomlCheckSyntax Module implementing the syntax check for TOML.
yamlCheckSyntax Module implementing the syntax check for YAML.
